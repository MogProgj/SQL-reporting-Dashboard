# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly by emailing the repository owner directly (see GitHub profile) rather
than opening a public issue.

## Supported Versions

Only the latest code on `main` is actively maintained.

## Scope

This is a small portfolio / learning project. It is not intended for production
use with real customer data. Nonetheless, contributions that improve security
(parameterised queries, secret handling, dependency updates) are welcome.
